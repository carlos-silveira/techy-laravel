# Plan Feature

Generate a detailed implementation plan for a new feature.

## Usage
```
/plan-feature FEATURE-NAME
```

## Steps

1. **Analyze the feature request** — Understand what needs to be built
2. **Identify affected files** — List all files that will be created/modified
3. **Create implementation plan** — Write to `ai-specs/changes/FEATURE-NAME.md` with:
   - Overview of the feature
   - Backend changes (models, controllers, services, routes)
   - Frontend changes (components, pages, API calls)
   - Database migrations (if needed)
   - Testing plan
4. **Request review** — Present the plan for approval before implementing
